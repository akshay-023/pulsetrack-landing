# PulseTrack Landing Page

Simple responsive landing page for the PulseTrack smartwatch.

**Deployed URL** (after setup): `https://akshay-023.github.io/pulsetrack-landing/`

## Files

- `index.html` — main page
- `style.css` — small custom styles
- `script.js` — interactive behavior

## Notes

This project is deployed using GitHub Actions to the `gh-pages` branch and served by GitHub Pages.
